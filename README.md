<h1>Environment variables</h1>

<p class=code><span class=GramE><span style='color:black;mso-color-alt:windowtext'>google.oauth2.clientId</span></span></p>

<p class=Textbody>The client-id provided by Google. The string I am looking at
is 72 characters long and ends with “apps.googleusercontent.com”.<span
style='font-size:12.0pt;line-height:115%;font-family:"Times New Roman",serif;
mso-fareast-font-family:"Times New Roman";mso-fareast-language:EN-GB;
mso-bidi-language:AR-SA'><o:p></o:p></span></p>

<p class=code><span class=GramE><span style='color:black;mso-color-alt:windowtext'>google.oauth2.clientSecret</span></span></p>

<p class=Textbody>The secret provided by Google. This string is 24 characters
long.</p>

<p class=code><span style='color:black;mso-color-alt:windowtext'>postgres.datasource.url</span></p>

<p class=Textbody>The <span class=SpellE>url</span> to get access to the <span
class=SpellE>postgress</span> database. </p>

<h1>Authentication</h1>

<p class=Textbody>The authentication takes place via google <span class=SpellE>Oauth</span>.</p>

<h1>Technical</h1>

<p class=MsoNoSpacing>This backend is made with Java, Spring Boot, JPA,
Hibernate, Swagger. The data is persisted in a Postgres database, unit tests take
place in a volatile H2 database.</p>
